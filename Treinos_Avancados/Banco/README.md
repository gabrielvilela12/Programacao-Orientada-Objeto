# Sistema de Gerenciamento de Contas Bancárias

Este projeto é um sistema de gerenciamento de contas bancárias desenvolvido em C#. Ele permite a criação de diferentes tipos de contas (contas comuns, contas empresariais e contas de poupança), além de realizar operações de empréstimos e atualizações de saldo.

## Funcionalidades

- **Criação de Contas**: Permite criar contas bancárias com diferentes características.
- **Empréstimos**: Permite que contas empresariais realizem empréstimos.
- **Atualização de Saldo**: Permite que contas de poupança atualizem seu saldo.
